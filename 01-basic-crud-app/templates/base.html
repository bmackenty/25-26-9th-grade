{# JINJA TEMPLATE ENGINE OVERVIEW FOR STUDENTS ================================================ This is the BASE TEMPLATE that all other pages inherit from. It demonstrates several key Jinja concepts: 1. TEMPLATE INHERITANCE: - Other templates extend this one using {% extends "base.html" %} - They can override blocks like title and content - This keeps navigation, styling, and common elements consistent 2. BLOCKS: - {% block title %} - Child templates can override this - {% block content %} - Child templates put their main content here - Think of blocks as "placeholders" that child templates fill in 3. INCLUDES: - {% include 'navbar.html' %} - Inserts another template file - Useful for reusable components like navigation menus 4. CONDITIONALS: - {% if condition %}...{% endif %} - Show/hide elements based on conditions - request.endpoint gives us the current page name 5. VARIABLES: - {{ variable }} - Display values passed from Flask - request.endpoint is a built-in Flask variable This template creates the overall page structure, and child templates fill in the specific content for each page. #} {# JINJA BLOCK EXAMPLE #1: TITLE BLOCK ===================================== This title block can be overridden by child templates. - If a child template defines {% block title %}, it will use that title - If no child template defines a title, it defaults to "Flask App" - This allows each page to have a unique title in the browser tab SYNTAX: {% block title %}Default Title{% endblock %} USAGE: Child templates can override with: {% block title %}My Page{% endblock %} #} {% block title %}Flask App{% endblock %} {# EXTERNAL LIBRARIES (Not Jinja-related) ======================================= Tailwind CSS: For styling (utility-first CSS framework) Alpine.js: For simple JavaScript interactivity #} {# JINJA INCLUDE EXAMPLE ===================== This includes the navbar.html template, which will be inserted here. Think of it like copying and pasting the navbar code into this spot. BENEFITS: - Reusable navigation across all pages - Easy to update navigation in one place - Keeps code DRY (Don't Repeat Yourself) SYNTAX: {% include 'template_name.html' %} LOCATION: The navbar.html file must be in the templates/ folder #} {% include 'navbar.html' %} {# MAIN CONTENT CONTAINER ======================= This div wraps all the main content of the page. - max-w-3xl: Maximum width of 48rem (768px) - mx-auto: Center the content horizontally - p-6: Add padding of 1.5rem (24px) on all sides #}
